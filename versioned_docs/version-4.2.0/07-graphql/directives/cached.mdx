whether this query should be cached (Hasura Cloud only)

```graphql
  refresh: Boolean! = false
)
```

### Arguments

#### [`ttl`](#) ([`Int!`](/07-graphql/scalars/int))

measured in seconds

#### [`refresh`](#) ([`Boolean!`](/07-graphql/scalars/boolean))

refresh the cache entry
